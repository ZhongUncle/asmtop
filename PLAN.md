# Mac 系统监控工具开发计划（C++版本）

## Context
用户需要一个高性能的工具来查看 Apple Silicon Mac 的系统数据，包括电池、功耗等相关信息。要求尽可能展示多样化的数据，并在终端中使用合适的方式展示（如条形图、折线图、饼图、进度条、表格等可视化形式）。已知可以使用 `pmset`、`system_profiler`、`powermetrics` 这几个 macOS 内置命令作为数据来源。**仅支持 Apple Silicon（ARM架构）的 Mac，无需兼容Intel架构**。
原Python版本已废弃，完全重写为C++版本，采用小步迭代式开发。

## 目标
1. 尽可能全面覆盖所有可获取的 Apple Silicon Mac 系统数据类型
2. 支持丰富的终端内可视化展示方式，直观呈现数据
3. 提供交互式命令行界面，支持多种数据视图切换
4. 实现实时数据更新和历史数据趋势展示
5. 仅针对ARM架构优化，充分利用Apple Silicon的特性
6. 极致性能：CPU占用<1%，内存占用<20MB，启动时间<100ms
