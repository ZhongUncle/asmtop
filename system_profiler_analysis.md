# system_profiler 命令分析报告

## 概述
system_profiler 是 macOS 上的系统信息报告工具，可以提供非常详细的硬件和软件配置信息。它支持文本、XML和JSON三种输出格式，非常适合程序解析。

## 核心数据类型

### 1. 电源相关信息 - SPPowerDataType
**输出内容**：
- **电池信息**：
  - 型号信息：序列号、设备名称、固件版本、硬件版本
  - 充电信息：电量百分比、是否充满、是否正在充电、电量警告状态
  - 健康信息：循环次数、电池状态（正常/建议维修）、最大容量
- **系统电源设置**：
  - AC电源和电池电源下的睡眠计时器设置
  - 休眠模式、低电量模式设置
  - 网络唤醒等功能状态
- **AC充电器信息**：
  - 连接状态、功率、ID、序列号、制造商、型号、软硬件版本
- **电源事件**：
  - 计划的唤醒/睡眠事件列表

### 2. 硬件基本信息 - SPHardwareDataType
**输出内容**：
- 设备型号名称（如 MacBook Air）
- 型号标识符（如 Mac17,3）
- 型号编号
- 芯片类型（如 Apple M5）
- CPU核心数（性能核/能效核）
- 总内存大小
- 系统固件版本
- 序列号、硬件UUID、UDID
- 激活锁状态

### 3. 内存信息 - SPMemoryDataType
**输出内容**：
- 内存大小
- 内存类型（如 LPDDR5）
- 制造商信息

### 4. 存储信息 - SPStorageDataType
**输出内容**：
- 磁盘总容量、可用空间
- 挂载点、文件系统类型
- 物理驱动器信息（设备名称、介质类型、协议、是否内置）
- S.M.A.R.T. 状态

### 5. 显示信息 - SPDisplaysDataType
**输出内容**：
- 内置显示器和外接显示器的详细参数
- 分辨率、刷新率、显示技术
- 显卡信息

### 6. 软件信息 - SPSoftwareDataType
**输出内容**：
- 系统版本（如 macOS 26.5）
- 内核版本
- 启动卷宗、启动模式
- 计算机名称、用户名
- 安全功能状态（安全虚拟内存、系统完整性保护）
- 系统启动时长

### 7. 网络信息 - SPNetworkDataType / SPAirPortDataType
**输出内容**：
- 网络接口配置
- Wi-Fi 连接信息、信号强度
- 蓝牙设备信息（SPBluetoothDataType）

## 输出格式支持
- **文本格式**：默认输出，层级清晰，适合人工阅读
- **XML格式**：使用 `-xml` 参数，适合程序解析
- **JSON格式**：使用 `-json` 参数（macOS 10.15+ 支持），非常适合现代编程语言解析

**JSON格式使用示例**：
```bash
system_profiler -json SPPowerDataType SPHardwareDataType
```

## 输出格式特点
- 结构化的层级输出，易于解析
- 支持一次性查询多个数据类型
- JSON格式输出可以直接反序列化为对象
- 数据全面，覆盖了系统硬件和软件的各个方面

## 推荐使用场景
1. 获取设备基本硬件配置信息（型号、芯片、内存、存储）
2. 查询电池详细健康状态（循环次数、最大容量）
3. 获取系统软件版本和配置信息
4. 查看外设连接状态（显示器、存储设备、网络设备）
5. 生成系统完整配置报告

## 性能注意事项
- 查询多个数据类型时可能需要几秒钟时间
- 建议按需查询所需的数据类型，避免不必要的性能开销
- JSON格式输出比文本格式更高效，适合程序使用
